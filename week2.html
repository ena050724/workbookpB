CRAZY EIGHTS

This week we started with Crazy 8s - ideas in a minute for our workbook website. When I shared mine, the underwater world concept got the best reaction in the room: an ocean where you have to catch the right creature to unlock each week content, like a game. That response was enough for me to commit to it. This website you’re on right now is that idea. The Tag activity was chaotic in a good way! Running around collecting HTML tag pieces (div, a, p…) to assemble a working webpage. Some tags were missing so we had to write them ourselves. I was slow and honestly still fuzzy on which tag does what, but moving physically helped me remember more than just reading ever would. Paper prototyping was my favourite part of the week. We got magazines, coloured paper, and pens to design a web interface by hand. My idea was a train: you piece together a puzzle to get a key, which unlocks each carriage representing a week, but some carriages would be empty, so watch out!
