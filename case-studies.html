Case Studies

A selection of experimental web projects exploring minimal layout, interactive reading systems, and playful useless interfaces.

tdingsun - Aayushi

This website uses a minimal layout with diverse interaction methods. Text can scroll vertically inside small blocks and slide horizontally, creating a reading experience similar to Japanese calligraphy panels, which makes navigation feel intentional and engaging.

JET SET WILLY Variations ©1984

The website is extremely minimal, using a strong yellow background and tightly arranged images. The compressed composition creates the feeling of a fragmented visual narrative, almost like reading a story through sequential screenshots.

Paint Toys – Oil

The title is striking and the site presents a collection of playful Japanese-style mini experiences such as stone stacking, pixel drawing, and scratch art. The interface feels intentionally useless but encouraging curiosity and exploration.

Jeffrey Deitch

https://deitch.com

This website feels visually rich and dynamic. The six image blocks are all moving, which makes the homepage immediately feel alive. When clicked, each image opens into more visual content, so the site feels like a gallery that is already in motion before you even enter.

Art.Teleportacia

https://art.teleportacia.org

I really like the idea behind this site. It starts from a tiny home button, but clicking it opens up endless little worlds. It feels intimate and exploratory, almost like reading someone’s diary through hyperlinks and discovering hidden thoughts page by page.

Simone Hoang

https://simonehoang.com

This website is very striking because each image starts blurred and only becomes visible when the cursor hovers over it. The left bar and top bar also move automatically, which makes the whole site feel controlled, elegant, and aesthetic. The interaction is simple, but very memorable.
